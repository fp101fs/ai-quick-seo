# Automate On-Page SEO with AI: Prompts and Tools for Real-Time Optimization

Your Search Console shows pages with strong impressions but weak engagement. That's a fixable problem. AI can help you optimize the on-page elements users actually see — title tags, meta descriptions, and content hooks — without requiring a full rewrite every time.

## Step 1: Target Critical On-Page Elements First

Not all on-page elements matter equally for CTR. Start here:

1. **Title tag** — the #1 factor in click decisions
2. **Meta description** — the supporting argument that seals the click
3. **First paragraph** — critical for bounce rate once they arrive

Use [Google Search Console](https://search.google.com/search-console) to find pages with high impressions but low CTR (under 3%). These are your immediate targets.

[SerpDo's Opportunities page](/opportunities) automatically flags these pages and generates copy prompts pre-loaded with your real query data — no manual CSV digging required.

## Step 2: Build Actionable AI Prompts

Generic prompts get generic results. Use your actual data:

**For title tags:**
```
This page targets "[keyword]" and currently gets [X] impressions but [Y]% CTR.
The current title is "[title]". Rewrite it to include [specific benefit users care about].
Make it under 60 characters. Include a number or year if relevant.
```

**For meta descriptions:**
```
The current meta description is "[text]". Users searching "[keyword]" want [specific outcome].
Rewrite the meta description to promise that outcome clearly. Under 155 characters.
Use active language and avoid passive constructions.
```

The specificity is what makes these prompts work. Vague prompts → vague output.

## Step 3: Test Continuously

Apply one change at a time. Track CTR daily in [Search Console](https://search.google.com/search-console). Wait 7–14 days before judging results — Google's data has a lag.

The fastest wins come from small, iterative changes rather than full page rewrites. Two tweaks per week beats one overhaul per month.

[SerpDo's Rank Tracking](/rank-tracking) shows position trends over time, so you can correlate on-page changes with ranking movements.

## Real Example

A client's "SEO Tools" page had this meta description: "Comprehensive guide to SEO tools." Accurate, but useless. No benefit, no specificity, no reason to click.

AI rewrite: "Get 5 free SEO tools to boost your rankings in 2024 — no signup required." CTR jumped from 1.5% to 4.1% in 7 days. Same page, same rankings, 2.7× more traffic.
